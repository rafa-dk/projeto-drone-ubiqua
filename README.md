# Controle de drone por gestos - Computação Pervasiva e Ubíqua

Este projeto consiste no desenvolvimento de um ecossistema de Computação Ubíqua e Pervasiva capaz de interpretar sinais de linguagem corporal (gestos das mãos) e convertê-los em comandos de movimentação em tempo real para um drone, indo além dos métodos tradicionais de controle (controles remotos físicos ou analógicos virtuais).

## 🛠️ Tecnologias e Componentes

### Hardware
*   **Drone:** Dispositivo inteligente principal para execução dos comandos de voo.
*   **Dispositivo Android:** Smartphone responsável por rodar o aplicativo de controle do drone.
*   **Câmera Externa:** Captura das imagens e do posicionamento dos gestos do usuário.

### Software e Ferramentas
*   **Android Studio:** Ambiente de desenvolvimento do aplicativo nativo.
*   **RCFPVPro:** Aplicativo base/protocolo de comunicação com o drone.
*   **MediaPipe (Gesture Recognizer):** Solução do Google utilizada para o reconhecimento e classificação dos gestos em tempo real.
